🗺️Mind Map
Um quadro interativo para organizar atividades da melhor forma que você quiser. No fundo, sempre quis ter um mapa mental desses pra deixar bagunçado, mas nunca vi uma aplicação de graça pra poder brincar com a ideia. Fiz ele de um jeito que você possa salvar diferentes mapas em .json e depois abrir eles, pois mesmo que haja bagunça, é uma bagunça organizada.
